# Robô Agressivo — Tablet (PWA)

Aplicativo web instalável para tablets Android.

## O que ele faz

- painel responsivo;
- sinal demonstrativo CALL, PUT ou NÃO OPERAR;
- pontuação técnica;
- gerenciamento de banca;
- stake percentual;
- stop diário;
- limite de perdas consecutivas;
- histórico de GREEN, RED e empate;
- funcionamento offline após a primeira abertura;
- dados salvos localmente no navegador.

## Limitação importante

Esta versão não recebe dados reais da Binomo e não executa operações.
A geração de sinais desta primeira versão é demonstrativa, destinada a validar
interface, gerenciamento e fluxo operacional no tablet.

## Como instalar no tablet

Para a instalação como aplicativo, os arquivos precisam estar hospedados em um endereço HTTPS.

Opções:
1. GitHub Pages;
2. Netlify;
3. Cloudflare Pages;
4. servidor próprio.

Depois de abrir o endereço no Chrome:
- toque no menu de três pontos;
- escolha "Adicionar à tela inicial" ou "Instalar aplicativo";
- confirme.

## Teste local

Também é possível abrir o arquivo index.html diretamente no navegador,
mas o modo instalável e offline funciona melhor quando hospedado em HTTPS.

## Próxima evolução técnica
